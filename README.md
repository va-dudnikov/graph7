# graph7 0.2.0-alpha.1

## Requirements
Library written on pure C and no special requirements.
You need only **scons** build automation tool. I use scons v3.1.1.

## Build of the library
Just enter:
```bash
scons
```

To build examples:
```bash
scons examples=true
```

To build tests:
```bash
scons tests=true
```
