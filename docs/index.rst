graph7 |release| documentation
==============================

Welcome to the documentation for version |release|.

graph7 is the C library that implements functions for working with different graph storage formats.
Namely, popular encoding formats graph6, sparse6 and digraph6
(`Brendan McKay <https://en.wikipedia.org/wiki/Brendan_McKay>`_,
specification see `here <http://users.cecs.anu.edu.au/~bdm/data/formats.txt>`_),
as well as encoding format graph7 (format specification will be given later).
